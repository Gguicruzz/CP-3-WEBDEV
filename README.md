#  Relógio Molezinha

Projeto desenvolvido para o **Checkpoint de Web Development (JavaScript)**.

##  Objetivos
- Praticar manipulação do DOM.
- Usar `setInterval()` para atualizar a hora em tempo real.
- Usar o objeto `Date` do JavaScript.

##  Funcionamento
O relógio mostra as horas, minutos e segundos do sistema e atualiza automaticamente a cada segundo.
